My Flix App

This web application will provide users with access to information about different movies including their directors, genres, lead roles and short descriptions. Users will be able to sign up, update their personal information, and create a list of their favorite movies!

My Flix API Documentation

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object holding an arrary with all movies.
Get data about a single movie by title /movies/:Title GET none A JSON object holding data about a single movie, containing description, genre, director, image URL, and cast.
Get data about a genre by genre's name /movies/genre/:genreName GET none
Get data about a director by director's name /movies/directors/:directorName GET none A JSON object holding data about a director. ExampleL { "Name":"Tony Kaye", "Birth":"1951-07-08" }
